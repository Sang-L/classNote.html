서비스공학 5주차 강의 노트

경쟁전략 :
1)원가우위전략 : 원가를 낮추어 경쟁우위를 확보
규모의경제를 확보해야함, 표준화/기술화/온라인서비스활용
주의사항 : 무조건 싸다고해서 좋은게 아님.(품질 대비 체감가격을 낮춰야함 -> 고객의 만족도), 브랜드의 이미지에 영향

2)차별화전략 : 경쟁사와 다른 독특한 서비스가 무엇인가?
ex)스타벅스.
무형적 요소를 유형화시키던가/ 서비스 정보를 사전에 제공한다던가..( ex)자동차 수리 -> 어떻게 수리를 할지 견적을 먼저 받고 하는 경우)

3)집중화전략 : 확실한 시장을 설정하고(ex허리디스크전문병원 등) 성격이 분명한 목표시장에 집중하는 전략.
차별화와 다른 성격임.


마케팅전략
1)STP전략수립, 4P마케팅믹스 결정
상황분석(시장/소비자분석) -> STP전략수립(세분화,표적시장,포지셔닝) -> 마케팅믹스결정(제품,가격,유통,촉진)

2)시장조사
-1) 설문조사법
-2) 면접조사법
-3) 패널조사(시간의흐름에따른분석가능)
-4) CLT (설문과 면접의 장점을 적당히 가지고있음)


3)소비자분석
구매행동/ 배분행동/ 사용행동의 분석
구매결정 과정분석(LFD의 결합 )

STP분석
S : 시장 세분화
T : 목표시장선정
P : 그 시장 내에서 어떤 서비스를 제공할 것인지(차별화필요) --> 브랜드의 이미지를 여기서 구축

4P분석 --> 확장된 개념인 7P(물리적 표식(설비의 설계, 종업원 복장 등), 프로세스(서비스활동의 흐름, 고객참여수준 등), 사람(종업원선발/교육) 의 개념을 추가함)
4C전략 (고객,비용,편의,소통) -->고객입장에서 바라보는 관점(고객의 니즈는 무엇이냐)

마케팅믹스의 구성단계
1단계 : 마케팅믹스를 구성하는 핵심요소 결정
2단계 : 각 요소의 정량화를 통해 가능한 대안 설정
3단계 : 목적함수를 극대화(이익)
P1(서비스특성수준), P2(서비스가격수준), P3(유통비용 수준), P4(촉진비용수준)
4단계 : 요소간 상호관계를 고려하여 전체적 믹스 조정


가격전략 :

1)점유율확대 vs 이익확대

2)
원가 -> 가격의하한선
고객 -> 가격상한
경쟁자 ->중간

3)
원가기반전략 : 원가 + 적정이윤으로 가격 책정->원가계산이 어려움
경쟁자기반전략 : 경쟁기업의 가격을 토대로 가격선정 -> 경쟁업체가 많지 않으면 경쟁자기번전략쓰면된다. 그러나 대체제(직/간접)에 대한 고민이 필요
고객기반전략 : 가격에 민감하게 반응하는 시장에 적합,

가격측정방식
1)고정가격법 : 모든 고객에게 동일한요금
2)차별가격법 : 고객에 따라 다른 요금(어린이요금 등)
3)다중가격법 : 서비스 품질에 따라 가격수준을 다르게 함(항공권 등)
4)동적가격법 : 수시로 다른 가격을 책정(음식점의 시가)
