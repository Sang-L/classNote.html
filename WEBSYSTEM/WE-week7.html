Classification 1)기본 개념 -1)분류(classification)란? 레코드(데이터)를 미리 정한 범주에 할당하는 것. 범주를 클래스라 부르고 클래스는 레이블로 구분 -2)레코드(X, Y) X : 속성 집합, 이산형 혹은 연속형(EX 학생) Y : 범주(category) 혹은 클래스 레이블, 이산형(EX 학생이 우등생이냐 아니냐) -3)분류는 목표 함수 f를 학습하는 작업 f : 데이터 X를 클래스 레이블 중 하나인 Y에 사상 (f : 데이터 X를 Y에 사상) -4)분류의 예 -암세포 예측 -스팸메일 필터링 -신용카드 합/불법 판정 2)목표함수(target function) -1)목표함수 -분류모델 이라고도 불림 -이진/명목형 범주를 가지는 데이터 집합을 예측/서술하는데 적합 -서열형에는 덜 효과적 -2)서술 모델링 서로 다른 클래스의 객체들을 구별하기 위한 설명 도구 -3)예측 모델링 클래스 레이블을 예측하기 위해 사용 3)분류 기법(분류기, classifer) 의사결정트리, 나이브베이지안, 인공신경망, 지지도 벡터기계, 앙상블 등등을 이용해 속성 집합과 클래스레이블 간의 관계에 적합한 모델을 찾는 것. 4)분류를 위한 일반적인 접근 방법 x는 무조건 훈련데이터와 시험데이터로 나눈다. -1)훈련데이터집합 -> 분류기로 분류 클래스레이블이 알려진 레코드들로 구성 -2)시험데이터집합 클래스레이블이 알려지지 않은 레코드들로 구성 -3)분류 모델 평가 -모델의 성능평가 -혼동 행렬 -다양한 평가 척도 Confusion matrix f11 : 실제 p 예측 p -> tp f01 : 실제 n 예측 p -> fp f10 : 실제 p 예측 n -> fn f00 : 실제 n 예측 n -< tn ex) 정확히 예측된 클래스 개수 : f11 + f00 부정확히 예측된 클래스 개수 : f01 + f10 accuracy : (f11 + f00)/(f11 + f00 + f01 + f10) Evaluation metric -평가척도 종류 --Accuracy --Precision(or, positive predicted vlaue)★ --Recall(or, true positive rate, sensitivity)★ --Specificity --FP rate --F1 score --Kappa --Balanced accuracy★ -1)평가척도란? -분류모델을 평가하는 수단(ex : accuracy : 분류 모델이 올바르게 예측한 비율, 가장 보편적 척도) Cross validation(교차 검정) -개념 : 분류 모델을 검정하는 수단
