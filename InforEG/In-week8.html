5장. 기술혁신

1.기술혁신의 개념
-1)혁신의 정의
혁신 - 기술혁신, 경영혁신으로 나뉜다.

-혁신이란?
에드 로버츠 : 좋은 제품을 발명하여 상업화에 성공하는 것.
마이클포터 : 기업의 혁신 행위가 경쟁력 확보의 핵심 요인
스티브잡스 : 창의성은 단순히 여러가지 요소들을 연결하는 것.


-2)기술혁신
-기술혁신의 정의
사전적정의 : 기술상에서 새로우 기술이나 과정이 도입되어 기존의 것과 확연히 달라지는 혁신이 일어나는 것. 생산기술의 획기적인 발전을 비롯하여 새로운 상품의 도입, 자원의 개척 및 새로우 경영조직의 도입 등이 일어나는 경제사회 구조의 변혁
학문적 정의 :
J.A Schumpeter : 공정, 시장, 재료 및 조직 등 생산수단의 새로운 결합을 통하여 신제품이나 서비스를 생산하고 마케팅 및 판매하는 일련의 현상
C.Freeman : 새롭거나 개선된 제품, 공정,장비를 상업적으로 활용하는 것과 관련된 기술, 디자인, 제조, 관리, 비즈니스 등에 대한 활동

-기술혁신의 특징
ㄱ)기술궤적
기술혁신이 한 번 일어나면, 이어지는 기술혁신은 어떤 방향으로 향하여 계속 움직임
ㄴ)누적적 속성
누적적 속성은 지식의 속성이라 할 수 있음. 즉 지식 학습량이 증가할 수록 기술혁신을 창출해낼 가능성도 누적적으로 높아짐
★기술혁신 능력을 배양하는 데 학습의 중요성 강조★
ㄷ)사회적 요인에 의한 영향
기술혁신과 사회는 상호적 연관이 있음.


2.기술혁신의 유형과 원천
J.A Schumpeter의 5가지 기술혁신 유형
-새롭거나 개선된 제품
-새롭거나 개선된 공정
-새로운 시장의 형성
-원재자 혹은 반제품에서 새로운 원료 확보
-산업 혹은 기업의 조직적 변화

※제품혁신
새로운 제품을 개발하거나 기존 제품의 성능 또는 기능을 향상시키는 것.
ex_ 테슬라 전기자동차.

※공정혁신
제품 생산과정에서 비용을 줄이는 공정을 새롭게 개발하거나 기존의 공정을 개선하는 것.
ex_키오스크(ATM기기, 맥도날드 무인주문기)

-제품혁신과 공정혁신의 관계
(1975년 UtterBack논문)
Y축 기술혁신의 비율
X축
유동기 : 제품혁신 높음, 공정혁신 낮음
과도기 : 공정혁신 높음, 제품혁신 낮음(계속낮아짐)
경화기 : 공정혁신높음(낮아지는형태), 제품혁신 낮음(공정혁신보다 낮음)
경화기에는 크게 이윤을 얻기 힘들어짐.


※기술혁신의 원천
-기술주도이론 : 기업이 기초과학 지식을 활용하여 이를 제품으로 만들려는 노력을 통해 기술혁신이 일어난다는 이론
-수요견인 이론 : 기초과학이나 기술 자체의 변화에서 오는 것이 아니라 소비자의 수요를 감안한 기업의 이윤 증대 기회와 연결될 때 이루어진다는 것을 강조

강조 새로운 욕구/수요 ---- 시장과 사용자의 욕구
| | |
아이디어생성 | ->개발 -> 생산 -> 마케팅/판매 -> 시장
| | |
새로운 기술적 <---> 최고/최초의 기술과 생산 기법
능력/기회



※혁신 주체 : 생산자 혁신에서 사용자 혁신으로
Producer Innovation -> User Innovation

생산자 혁신과 사용자 혁신의 특성비교
-생산자혁신(제조업체)
혁신주체 : 생산자
특성 : 대규모투자, 생산원가 절감이 목표
장점 : 안정적 생산 및 판로
단점 : 고객 니즈에 대한 구체적인 정보 부족

-상용자혁신
혁신주체 : 사용자(고객)
특성 : 특정 고객 및 구체적 니즈 충족, 혁신에 대한 혜택을 직접 수혜
장점 : 다양한 고객 니즈 충족
단점 : 판로 개척의 어려움, 사용자 요구의 이절성이 높은 경우 수용하기 어려움, 디자인의 견고성 부족 등

※혁신 분야 : 제조업 혁신에서 서비스업 혁신으로
-우리나라를 비롯해 주요구에선 1인당 국민소득이 늘어나면서 서비스산업이 경제 성장을 견인하는 '경제의 서비스화'가 진행
-산업 방향이 제조업에서 서비스업으로 변화하고 제조업과 서비스업의 융화가 일어남에 따라, 기술혁신도 제조업 혁신에서 서비스업 혁신으로 변화

※혁신 공유 : 폐쇄형 혁신에서 개방형 혁신으로
-개방형 혁신
--연구,개발,상업화에 이르는 기술혁신의 모든 과정에서 대학이나 타 기업, 연구소 등의 외부 기술이나 지식, 아이디어를 활용한 혁신
--개방형 혁신을 내방형과 외향형으로 구분하여 제시
-> 내향형 : 기술구매, 공동연구, 연구개발 등
-> 외향형 : 기술판매, 분사(스핀오프)

※혁신 활동 : 네트워크에 기반한 혁신 활동
-정보통신 기술이 발전함에 따라 온라인 형태로 혁신 활동에 참여하는 사례가 늘고 있음.
-혁신공동체
--혁신 활동에 참여하는 공동체
--비슷한 목적을 가진 전문가들이 모여 상호 학습을 하는 장으로 활용
--정보통신 기술을 도구로 활용하면서, 혁신공동체의 활용 및 활동 범위는 더욱 커지고 있음.
-->네트워크가 공동체를 활용하여 혁신 활동에 활용하는 사례가 크게 증가할 것이며 활용 범위도 매우 다양해질 것으로 전망
ex) 5G 기반 스마트건설 혁신 네트워크

※혁신의 방식 : 정형화된 혁신에서 비 정형화된 혁신으로
-전통적 기술혁신 성과는 연구개발 활동을 통해 산출된 성과물로 특허를 받아 신제품을 만들어 시장에 출시하는 것.
-전 세계적으로 특허를 개방하고 이를 활용하려는 움직임이 확산 -> 특허 등록이 아닌 R&D 활동 결과물 공유
