2)상자그림(box plot)
※박스안의 데이터는 전체의 50%에 해당
IQR = Q3 - Q1, Q1,Q2,Q3의 차이알기, Lower Limit(Q1 - 1.5IQR)->★피피티 자료 오타남★, Upper Limit(Q3 + 1.5IQR) --> 모든 데이터는 Upper and Lower 내에 존재해야 함. 그 밖의 데이터는 '이상치' 라고 함.
박스 바깥으로 대칭
장점 : 이상치를 찾아낼 수 있음. -> 6시그마에서 히스토그램과 마찬가지로 많이 사용(이상치를 찾아내기위해) 미니탭 실습 : 11:30

1)계통도
목적->수단->목적->수단->목적->수단 ----~~ 원래의 목적을 달성하기 위한 방법이 나타날 때 까지 전개하는 것.
용도 : 일차적 목적이나 프로젝트 완수를 위해 하위 단계들을 논리적으로 전개.

2)네트워크도 : 여러가지 복잡한 순서를 목적이 달성될 때 까지 작업순서와 시간 배정을 나타낸 것.
목적 : 프로젝트의 모든 활동들의 선후관계를 밝힌다.
-1)화살표도형법 : 프로젝트의 모든 활동을 화살표로 표시. 착수와 완료시점은 마디로 표현.
'마디와 노드'가 존재.
-2)선후행도형법 : 마디위에 활동이 들어감.
-->화살표도형,선후행도형 중 선택해서 사용
※주공정법(Critical Path Method)의 예시) 7분 정도에 설명
네트워크도 적용절차 -> 단계1~4 참고
네트워크도 적용사례 -> 공항 도착->출국까지의 과정.

3)매트릭스도 : 두 개 또는 그 이상의 특성, 기능, 아이디어 등의 집합에 대한 관련 정보를 행렬로 표현
★L,T,X,Y 형 파악, 기호 파악★
매트릭스도 적용 절차 생략(걍 읽어봐)
4)매트릭스 데이터 해석도 : 마케팅분야에서 제품이나 서비스의 포지셔닝을 결정하기 위해 자주 사용.(MDS 다차원척도법 적용)
